Starter prompt:

Check whether ai-skill-toolkit is already applied to the current project. If it is not applied, apply it.

Rules:
1. If the project root contains an ai-skill-toolkit folder, use scripts/install-to-project.ps1 from that folder.
2. If it does not exist, clone this repository:
   https://github.com/rkalf00-ship-it/ai-skill-toolkit.git
3. Run the installer from PowerShell to copy/install the skills into the current project:
   powershell -ExecutionPolicy Bypass -File ".\ai-skill-toolkit\scripts\install-to-project.ps1" -ProjectPath "."
4. Verify these files/folders were created:
   - .agents/skills
   - .claude/skills
   - .codex/skills
   - AGENTS.md
   - CLAUDE.md
5. If AGENTS.md or CLAUDE.md already exists, do not overwrite blindly. Compare differences and merge only the necessary content.
6. Summarize the installed skills and briefly explain when to use each one in the current project.
7. In later work, follow AGENTS.md and CLAUDE.md first.
